## 🎯 Objetivo do Repositório

Este repositório tem como objetivo armazenar as atividades desenvolvidas na disciplina de Qualidade de Software, utilizando a metodologia PBL.

## 📚 Organização do Projeto

* `docs/` → Contém as atividades realizadas durante o semestre
* `artefatos/` → Diagramas, relatórios e evidências das atividades
* `src/` → Código-fonte dos projetos
* `referencias/` → Materiais de apoio utilizados

## 📦 Entregas

As atividades estão organizadas dentro da pasta `docs/`, separadas por número de atividade.

## 🧩 Metodologia

O projeto segue a metodologia PBL, com foco na resolução de problemas práticos relacionados à qualidade de software.

## ✅ Boas práticas adotadas
