## Análise de Qualidade (ISO/IEC 25010)

| Problema identificado | Atributo de qualidade afetado | Justificativa técnica |
|----------------------|------------------------------|----------------------|
| Lentidão em horários de pico | Eficiência de Desempenho | Tempo de resposta elevado sob alta carga prejudica a experiência do usuário |
| Telas confusas e pouco intuitivas | Usabilidade | Interface mal estruturada dificulta a navegação e aumenta a ocorrência de erros |
| Buscas retornam resultados incorretos | Adequação Funcional | O sistema não entrega resultados corretos conforme esperado |
| Aplicativo falha em alguns smartphones | Compatibilidade | O sistema não funciona corretamente em diferentes dispositivos |
| Dificuldade para concluir ações simples | Usabilidade | Fluxos complexos dificultam a execução de tarefas básicas |
| Avaliações desaparecem após atualização | Confiabilidade | Falha na persistência de dados compromete a integridade das informações |
| Inconsistência entre web e mobile | Compatibilidade | Diferenças entre plataformas geram inconsistência na experiência do usuário |
